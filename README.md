# Public Goods Game

## Live site
[grumman-publicgoods.herokuapp.com/](https://grumman-publicgoods.herokuapp.com/)

## To start locally

```
git clone git@github.com:NoahGrumman/PublicGoodsGame.git
cd PublicGoodsGame
otree resetdb
otree runserver
```
